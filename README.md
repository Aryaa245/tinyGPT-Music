# tinyGPT-Music
A minimalist, implementation of the Generative Pre-trained Transformer (GPT) architecture. This repository is specifically designed as a practical guide to learning and understanding the inner workings of Large Language Models (LLMs).

# TinyGPT Music Experiment

Eksperimen TinyGPT menggunakan corpus bertema musik dan lagu.

## Corpus

- corpus.txt

Jumlah kata:
- ±2500 kata

### 2. Unigram

### 3. Character
